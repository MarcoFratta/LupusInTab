How It Works

1

Local Game State

Game data is stored locally on your device only, never transmitted anywhere

2

No Network Requests

The app works completely offline and doesn't make any network calls

3

Complete Privacy

Your game sessions remain completely private and under your control

Technical Details

This application is built as a Progressive Web App (PWA) that runs entirely in your browser. All game logic, player data, and game state are processed locally using JavaScript. No data leaves your device at any point during gameplay or between sessions.
